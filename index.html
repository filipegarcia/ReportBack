Report Tool

Description

The Tool is loaded dynamicly when clicking the feedback button. It grabs user and browser information and the page html code and screenshot.
It allow the user to highligt or blackout a portion of the page.
The screenshot is done on client side and allows to see the viewport of the client and how the page was rendered. For this html2canvas comes to the rescue. Taking the screenshot in client side is like buiding a browser inside the browser, and render the page that browser.
In the end, you can save the result that was built in json format
