Report Back

Description

The Tool is loaded dynamically when clicking the feedback button ( not implemented yet :) ). It grabs user and browser information and the page html code and screenshot.
It allow the user to highlight or blackout a portion of the page.
The screenshot is done on client side and allows to see the viewport of the client and how the page was rendered. For this html2canvas comes to the rescue. Taking the screenshot in client side is like building a browser inside the browser, and render the page that browser.
In the end, you can save the result that was built in json format

Markup

<script type="text/javascript">

  warmup();

</script>

What is grabbed?

Browser info
Version
Installed plugins
Screenshot
a rendered version of the page with highlight and blackout parts
cropped version of the viewport
Page info
Full url info
page structure
User info
Cookies
everything that
Anything else
just add more stuff to be recored from you own system

Dependencies

They can be all injected in the page when the user clicks the feedback button.
If you are already using any of these dependencies, you can remove them from the load 

<link href="assets/css/bootstrap.min.css" rel="stylesheet">
<link href="assets/css/jquery.ui.css" rel="stylesheet">

<script src="assets/js/jquery.js"></script>
<script src="assets/js/bootstrap.min.js"></script>
<script src="assets/js/jquery.ui.js"></script>
<script src="assets/js/html2canvas.min.js"></script>
